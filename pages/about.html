About

The website is not to serve as a professional site to introduce Hubble Space Telescope,you can refer to https://hubblesite.org or https://science.nasa.gov/mission/hubble instead. It is my personal project to practice using markdown, HTML, and AI chatbot to design a complete website. The content is pretty much from AI chatbot, which was generated from prompts. Because the content from AI chatbot is mostly Markdown format, not in the HTML format for a web brower to browse. I need to convert the markdown text to HTML format with Marktext application https://github.com/marktext/marktext. It is noted that I don't apply for a domain name. So I can demo locally or I put them to a third public space with subdomain. Overall, I think it is very interesting to learn website development in this way. I heard that today's AI technology can generate a complete website directly with some AI chatbot. I will dig more if there are chances in future. .
